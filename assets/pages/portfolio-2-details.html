Project Summary

Having coded in other languages, such as Powershell and Python in the past I've already had the confidence in my own ability to create simple games. I used this project as a way to translate my previous coding experience into Javascript. While the coding logic didn't give me much difficulty, I decided to take it a step further and add a theme to one of my favorite childhood games "Final Fantasy". The hardest part of this project was getting the animation and styling down.
